FarmaSystem 



FarmaSystem es un proyecto desarrollado en C++ que simula un sistema básico para la gestión de una farmacia.

La idea principal del proyecto es practicar programación y aplicar conceptos vistos en clase, como el uso de funciones, arreglos, punteros y organización del código.



Este proyecto también sirve para aprender a utilizar Git y GitHub para llevar control de versiones durante el desarrollo del programa.



Objetivo



El objetivo de este proyecto es desarrollar un programa que permita manejar información básica de una farmacia, como el registro de productos y algunas operaciones simples relacionadas con ellos.
